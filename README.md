## Description

Kidney - Water Tracker is an app developed in Android Studio (Kotlin) with a focus on encouraging daily water consumption in a simple, fun, and visual way.
It has an intuitive interface, motivational messages, a kidney-shaped mascot that changes appearance as you drink more water, and extra intake suggestions on hot days.
